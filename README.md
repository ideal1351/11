## Результаты сравнения размеров образов

| Сервис | Базовый образ | Размер |
|--------|---------------|--------|
| Go (scratch) | scratch | ~8 MB |
| Python (slim) | python:3.12-slim | ~180 MB |
| Rust (musl + alpine) | alpine:latest | ~7 MB |

### Анализ

Go и Rust образы значительно меньше Python образа по следующим причинам:

1. **Статическая компиляция** – Go и Rust компилируются в машинный код без внешних зависимостей
2. **Минимальные базовые образы** – scratch (0 байт) и alpine (~5 MB) против полноценного Python
3. **Отсутствие интерпретатора** – Python требует установки интерпретатора и runtime
