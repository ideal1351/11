РЕЗУЛЬТАТЫ СРАВНЕНИЯ РАЗМЕРОВ ОБРАЗОВ
========================================

| Сервис | Базовый образ | Размер |
|--------|---------------|--------|
| Go (scratch) | scratch | ~8 MB |
| Python (slim) | python:3.12-slim | ~180 MB |
| Rust (musl + alpine) | alpine:latest | ~7 MB |

Вывод: Go и Rust образы значительно меньше Python образа из-за:
1. Статическая компиляция без зависимостей
2. Минимальные базовые образы (scratch, alpine)
3. Отсутствие интерпретатора

Детали:
- Go: статический бинарник, CGO_ENABLED=0, финальный образ scratch
- Rust: сборка с musl, статическая линковка, финальный образ alpine
- Python: требуется интерпретатор и зависимости (FastAPI, uvicorn)
